# POM_1manufactura
Automated testing project for a diploma work

1. Необходимо установить:
pip install dotenv

3. Для нормальной работы тестов вам потребуется создать в корне проекта текстовый файл .env в котором вы будите хранить свой логин и пароль в формате:
LOGIN=здесь_ваш_логин
PASSWORD=здесь_ваш_пароль

4. В .gitignore добавьте файл .env чтобы ваши данные не попадали в репозиторий
